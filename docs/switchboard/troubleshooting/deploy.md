# Deployment Failures

This page will document deployment diagnosis.

## Planned Coverage

- Preflight failures.
- Missing Acurast IPFS upload credentials.
- Quote or funding mismatch.
- Acurast submission and assignment failures.
- Local timeout after remote progress.
- `switchboard deploy status`.
- `switchboard deploy doctor`.
- `switchboard deploy resume --yes` guardrails.

Diagnosis should name the failing stage before suggesting recovery.
