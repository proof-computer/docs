# Deployment Lifecycle

This page will document the lifecycle from local intent to public route.

## Planned Stages

1. Local project and context selection.
2. Preflight and quote inspection.
3. Acurast job deployment.
4. Hub quote funding.
5. Job-signed registration.
6. DNS and job-owned certificate readiness.
7. Gateway route activation.
8. Validator evidence and settlement windows.
9. Status, renewal, replacement, or recovery.

The lifecycle docs should make dry-run, spend, activation, and fulfillment
boundaries easy to scan.
