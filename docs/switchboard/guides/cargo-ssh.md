# Cargo SSH Jobs

This page will cover the SSH-over-Cargo workflow once it is ready for public
developer documentation.

## Planned Coverage

- `switchboard init --template ssh`.
- Acurast Shell processor requirements.
- Dropbear and job-owned certificate behavior.
- Public SSH over SNI.
- `switchboard deploy doctor --probe`.
- `switchboard-cargo-bridge-doctor` for safe bridge diagnostics.

Do not present this as the default webserver quickstart. It is an advanced
workflow for inspectable Script/Cargo jobs.
