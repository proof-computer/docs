# Operator Preview

Operator documentation is included as a preview because the first public
Switchboard launch is PROOF-operated.

## Planned Pages

- Gateway runtime overview.
- Capacity and capability reporting.
- Route-state health.
- Day-two monitoring.
- Diagnostics that are safe for operators to run.

Operator pages must keep PROOF-owned DNS, ACME, recorder, relayer, and control
plane credentials out of operator-facing instructions unless a deliberate
authority change is accepted.
