# Config Reference

This page will document Switchboard configuration files and environment
variables.

## Planned Coverage

- `switchboard.json`.
- Local context files under the user home directory.
- `.switchboard/state.json`.
- Secret environment variable names versus secret values.
- Manifest and service-catalog overrides.
- DNS provider configuration.
- Runtime environment fields delivered to Acurast jobs.

The docs should clearly separate local developer config, public runtime config,
and PROOF/operator-only configuration.
