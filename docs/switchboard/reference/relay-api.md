# Relay API

This page will document public relay and control-plane APIs that developers or
jobs can call directly.

## Planned Coverage

- Network manifest discovery.
- Deployment intent readback.
- Observability readback.
- Job registration and certificate request signing boundaries.
- Customer hostname diagnostics.
- Error shapes and retry expectations.

Secret-bearing calls must require HTTPS in public examples.
