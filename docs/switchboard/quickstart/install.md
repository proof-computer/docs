# Install And Configure

This page will become the canonical installation and first-context guide.

## Planned Coverage

- Node.js and package-manager expectations.
- `npm install --global @proof-computer/proof-cli`.
- `proof plugins install @proof-computer/proof-cli-switchboard`.
- Standalone `switchboard` install and upgrade path.
- Required Acurast, Polkadot Hub, and private-beta credentials.
- Context files storing environment variable names rather than secret values.
- DNS provider configuration for the current private-beta DNS path.

## Acceptance Goal

A developer should finish this page with a selected Switchboard context and no
live credentials written into the project repository.
