# Quickstart

This section will hold the shortest supported path from an empty machine or
project directory to a public Switchboard HTTPS endpoint.

## Content Targets

- Install the public `proof` CLI and Switchboard plugin.
- Install or upgrade the standalone `switchboard` compatibility CLI.
- Configure a Switchboard context without committing secret values.
- Run `switchboard preflight --quote`.
- Launch the supported demo with `switchboard launch-demo`.
- Deploy a webserver project with `switchboard deploy`.
- Read status, logs, deployment reports, and recovery hints.

The first full content migration should port the current hello-world flow into
this section and keep commands fish-compatible where shell syntax matters.
